# App Settings

This guide covers the settings available in KaGet.

## Accessing Settings

From any screen, tap the **menu icon** (or your profile) and select **Settings**.

## Currency

Your currency setting controls how all monetary amounts are displayed throughout the app.

### Changing Currency

1. Go to **Settings**
2. Tap **Currency**
3. Select your new currency from the list
4. Tap **Save**

### What Changes

| Aspect               | Behavior                                               |
| -------------------- | ------------------------------------------------------ |
| **Symbol**           | Updates to new currency symbol (e.g., $ → €)           |
| **Formatting**       | Decimal separators and grouping adjust based on locale |
| **Existing amounts** | Values remain unchanged — no conversion                |

> **Important:** Changing your currency does **not** convert existing transaction amounts. A record of "100" stays "100" — only the display symbol changes.

### Example

| Before (USD) | After (EUR) |
| ------------ | ----------- |
| $1,234.56    | €1.234,56   |

The underlying value is still `1234.56`. Only the display format changes.

### When to Change Currency

- You've relocated to a different country
- You selected the wrong currency during onboarding
- You want to track finances in a different base currency

> **Tip:** If you need to track multiple currencies, consider creating separate wallets for each currency and using the currency name in the wallet title (e.g., "Bank Account (EUR)").
