# Goals

Goals help you save toward a target amount—the opposite of spending budgets. Instead of decreasing as you spend, goals increase as you contribute.

## Overview

| Aspect         | Budget              | Goal                       |
| -------------- | ------------------- | -------------------------- |
| **Purpose**    | Control spending    | Save for something         |
| **Direction**  | Balance decreases   | Balance increases          |
| **Completion** | Balance = 0 (spent) | Balance = target (reached) |
| **Spending**   | Allowed immediately | Locked until reached       |

## Creating a Goal

1. Go to **Budgets** from the main menu
2. Tap **Create Budget**
3. Toggle **Goal** mode (instead of Budget)
4. Select the **wallet** to save from
5. Enter a **name** (e.g., "Vacation Fund", "New Laptop")
6. Enter the **target amount**
7. (Optional) Add an **initial contribution**
8. Tap **Create**

> **Example:** Saving for a $1,000 vacation
>
> 1. Create goal named "Vacation Fund"
> 2. Set target: $1,000
> 3. Initial contribution: $200
>
> Result: Goal shows $200 / $1,000 (20% progress)

## Contributing to a Goal

Add money toward your goal over time:

1. Open the goal
2. Tap **Add Funds**
3. Enter the **amount** to contribute
4. Tap **Confirm**

The amount is transferred from your wallet to the goal.

### Contribution Rules

| Rule                           | Description                                 |
| ------------------------------ | ------------------------------------------- |
| Cannot exceed wallet balance   | You can only contribute what's available    |
| Cannot exceed remaining target | Contributions stop at 100%                  |
| From source wallet only        | Contributions come from the original wallet |

## Goal Progress

Your goal displays:

| Element             | Description                   |
| ------------------- | ----------------------------- |
| **Current balance** | How much you've saved         |
| **Target**          | The goal amount               |
| **Progress bar**    | Visual percentage toward goal |
| **Remaining**       | How much more to save         |

## Reaching Your Goal

When your goal balance equals the target amount:

🎉 **Goal reached!**

You now have options:

### Option 1: Spend It

The goal becomes spendable like a regular budget:

1. Open the reached goal
2. Tap **Add Expense**
3. Spend toward what you saved for

After spending, the goal follows normal budget rules (archives when balance = 0).

### Option 2: Release to Wallet

Return the saved amount to your wallet:

1. Open the reached goal
2. Tap **Release to Wallet**
3. Confirm

The full amount returns to your wallet, and the goal is archived.

## Goal States

| State              | Balance  | Can Contribute | Can Spend | Can Release      |
| ------------------ | -------- | -------------- | --------- | ---------------- |
| **In Progress**    | < target | ✅ Yes         | ❌ No     | ✅ Yes (cancel)  |
| **Reached**        | = target | ❌ No          | ✅ Yes    | ✅ Yes           |
| **Spent/Released** | = 0      | ❌ No          | ❌ No     | ❌ No (archived) |

## Canceling a Goal

Changed your mind? You can cancel at any time:

1. Open the goal
2. Tap **Refund**
3. Choose amount (or "Refund All")
4. Tap **Confirm**

The amount returns to your wallet. If you refund everything, the goal is archived.

## Tips

- **Set realistic targets** — Break large goals into milestones if needed
- **Contribute regularly** — Small consistent contributions add up
- **Celebrate reaching goals** — That's the point!
- **Use descriptive names** — "Europe Trip June 2026" beats "Travel"

## Examples

### Short-term Goal

> **New headphones** — Target: $150
>
> Contribute $50/month for 3 months → Goal reached → Spend on headphones

### Long-term Goal

> **Emergency fund** — Target: $5,000
>
> Contribute $200/month → After 25 months → Goal reached → Keep as safety net (don't spend!)
