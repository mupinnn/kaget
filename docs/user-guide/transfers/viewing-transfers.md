# Viewing Transfers

This guide explains how to find and understand your transfer history.

## Finding Transfers

### From the Transfers Menu

1. Tap **Transfers** from the main menu
2. See all transfers, newest first
3. Filter by date range if needed

### From a Wallet or Budget

1. Open the wallet or budget
2. Look for the **Transfers** section or tab
3. See transfers involving that account

## Understanding Transfer Records

Each transfer appears **twice** in the system—once for each account involved:

| View                | Shows As                              |
| ------------------- | ------------------------------------- |
| Source account      | **Outgoing** transfer (money left)    |
| Destination account | **Incoming** transfer (money arrived) |

Both records are linked and show the same information, just from different perspectives.

## Transfer Details

Tap any transfer to see:

| Field      | Description                    |
| ---------- | ------------------------------ |
| **Amount** | How much was transferred       |
| **Fee**    | Transaction fee (if any)       |
| **From**   | Source account name            |
| **To**     | Destination account name       |
| **Date**   | When the transfer occurred     |
| **Note**   | Your description (if provided) |

## Transfer History After Account Deletion

If you delete a wallet, its transfers are also deleted. However, the **other side** of each transfer retains a snapshot of the deleted account's name.

> **Example:** You delete "Old Bank Account" which had transfers with "Savings."
>
> - Transfers in "Old Bank Account" → Deleted
> - Transfers in "Savings" → Still visible, show "Old Bank Account" as source/destination name

## Identifying Transfer Fees

When a transfer has a fee, you'll see:

1. **On the transfer:** Fee amount displayed
2. **In source account records:** An expense record for the fee

This ensures your total expenses accurately reflect transfer costs.

## Filtering and Searching

You can filter transfers by:

| Filter         | Description                                  |
| -------------- | -------------------------------------------- |
| **Date range** | Transfers within a specific period           |
| **Account**    | Transfers involving a specific wallet/budget |
| **Type**       | Incoming only or outgoing only               |

## Common Questions

### Why do I see the same transfer twice?

You don't—you see it from two perspectives. When viewing all transfers, each transfer shows once. When viewing a specific account, you see only transfers involving that account.

### Can I export transfer history?

Check **Settings > Export** for data export options.

### How do I fix a wrong transfer?

Create a new transfer in the opposite direction to reverse it. Transfers cannot be edited or deleted.
