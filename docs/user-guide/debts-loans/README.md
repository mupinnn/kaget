# Debts & Loans

Track money you've borrowed or lent in KaGet.

## Overview

Debts & Loans helps you remember outstanding money:

- **Debts** — Money you borrowed (you owe someone)
- **Loans** — Money you lent (someone owes you)

Each debt or loan is linked to your wallet and automatically tracks balance changes.
